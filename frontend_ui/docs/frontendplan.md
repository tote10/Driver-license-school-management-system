General Recommendations for All Dashboards (SRS 4.5 & 5)

Role-based access — Each dashboard must only show what the role is allowed to see.
Notifications panel (top right or side) — Schedule changes, results, approvals (common for all roles).
Search bar — Quick search for students/instructors (especially useful for Supervisor & Manager).
Responsive design — Mobile-friendly as per non-functional requirements.
Audit log — Manager (and optionally Supervisor) can have a small recent logs section.
